# KDS Dashboard REST API (MVC)

**Ders:** Sunucu Tabanlı Programlama  
**Konu:** MVC Mimarisi ile RESTful API Tasarımı  
**Teknoloji:** Node.js (Express) + MySQL  

## Proje Amacı
Bu proje, KDS (Karar Destek Sistemi) senaryosu üzerinden **MVC mimarisine uygun** ve **REST prensipleriyle uyumlu** bir API geliştirmek amacıyla hazırlanmıştır.

## Senaryo
Sistem; ürün geri bildirimlerini (feedback) ve konularını (topics/sub_topics) analiz ederek;
- topic/ürün/departman bazlı şikayet dağılımları
- işletme geneli risk seviyeleri
- Skorlara göre karar önerileri

üreten bir karar destek API’si sağlar.
